诚聘英才

美工设计


职位描述:

岗位工作职责描述:
1、负责公司电子商务平台相关业务的开发与维护
2、负责核心模块的研发及设计
3、带领电子商务平台研发团队，使电子商务开台工作顺利开展

岗位要求：
基本要求：
1.计算机或相关专业；
2.身体健康，积极进取，诚实正直、勤奋、踏实，责任心强；
3.良好的学习、沟通、适应能力；
4.良好的团队协作精神和服务意识。

专业要求：
1、具备两年以上ASP.NET 开发经验
2、精通C#、ASP.NET，对 .Net Framework 有深入了解，熟悉LINQ更佳；
3、精通 SQL Server 数据库
4、精通JavaScript、Ajax，熟悉 jQuery更佳
5、熟悉面向对象分析与设计，设计模式

诚聘英才

  1. 高级.net工程师
  2. .net开发工程师
  3. 美工设计
